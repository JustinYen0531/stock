# 山體材質工程手冊 (Mountain Terrain Material Manual)

## 1. 背景漸層 (Background Gradient)
*   天空底層：`#0A0F1E` 
*   夜幕邊緣：`#0D1829`
*   山體交接：`#111F35`

## 2. 地形渲染架構 (Terrain Rendering Architecture)
### Level 1: Solid Fill (實體填充)
*   根據產業主題定義的基礎填充色（如 Intel 的深藍色）。

### Level 2: Pattern (底紋疊加)
*   透明度：`0.45 ~ 0.75`。
*   Composite Mode: `overlay` (增加深度感與細節)。
*   支援動態滾動的 Texture 快取以提升效能。

### Level 3: Sheen & Sheen (高亮層)
*   山脊線的反光處理，營造雪山或數位晶圓的光亮邊緣。

## 3. 數據驅動調整 (Data-Driven Tweaks)
*   `Volumn` (成交量) 反映在 Texture 的亮點密度上。
*   `Volatility` (波動率) 反映在材質的破碎度或位移速度上。
