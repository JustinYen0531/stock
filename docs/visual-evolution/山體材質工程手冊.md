# 山體材質工程手冊 (Mountain Terrain Material Manual)

## 1. 背景漸層 (Background Gradient)
*   深色底色：`#0A0F1E` 
*   夜空冷調：`#0D1829`
*   混合邊緣：`#111F35`

## 2. 地形材質層 (Layer Architecture)
### Level 1: Solid Fill
*   基礎實色填充。

### Level 2: Pattern Tile (底紋)
*   透明度：`0.45 ~ 0.75`。
*   Composite Mode: `overlay` (如果是發光主題)。

### Level 3: Sheen (光澤層)
*   漸層光影營造反光感。

## 3. 性能快取 (Cache System)
*   使用 `terrainPatternCache` 儲存已生成的像素 Tile。
*   切換主題時必須清空快取。
