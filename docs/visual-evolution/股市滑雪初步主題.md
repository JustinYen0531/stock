# 股市滑雪初步主題 (Initial Stock Ski Theme Mapping)

## 1. 核心映射邏輯
將股票代碼、公司名稱與產業類別映射為遊戲主題包。

## 2. 映射規則
*   **技術型 (Tech)**：芯片、AI、晶圓。特色：發光電路圖、冷調青色。
*   **能源型 (Energy)**：發電與綠能。特色：電能感、高亮閃爍。
*   **消費型 (Consumer)**：零售與物流。特色：溫暖調、包裹圖騰。
*   **金融型 (Finance)**：銀行、保險。特色：石柱感、古典古典色。

## 3. 三大主題模組
1.  `Environment Biomes`：決定天空、遠山與整體天空色調。
2.  `Company Props`：品牌特色障礙物。
3.  `The Mixer`：控制視差捲動與材質飽和度。
