# 股市滑雪初步主題 (Initial Stock Ski Theme Mapping)

## 1. 核心定義與工作流
將股票代碼、公司名稱、產業類別與市場主題映射成遊戲主題包。

## 2. 映射規則
*   **數據決定幻想**：優先採用 Sector 或 Industry 資訊。例如：Semiconductors 則對應數位電路風。
*   **氛圍優先**：決定玩家前兩秒的感受（速度、壓迫、精密、工業）。
*   **產業導向範例**：
    *   `Energy`：發電、儲電、再生能源。暗金配色。
    *   `Consumer`：零售與物流。暖色紙質風格。
    *   `Tech`：晶圓、電路、AI。青色數位導線風。
    *   `Finance`：古典建築。石色莊嚴風格。

## 3. 主題四大模組
*   `Environment Biomes`：大氣、天氣、色調。
*   `Company Props`：品牌特色符號。
*   `The Mixer`：視差捲動與材質結合。
*   `Implementation`：Canvas 實作。
