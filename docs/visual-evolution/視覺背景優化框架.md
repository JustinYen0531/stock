# 視覺背景優化框架 (Visual Background Optimization Framework)

## 1. 視差捲動定義 (Parallax Standards)
*   **遠景 (Far Vista)**: `terrainScrollX * 0.1` 
*   **中景 (Mid Range)**: `terrainScrollX * 0.35`
*   **主地形 (Terrain)**: `terrainScrollX * 1.0`

## 2. 三層景深層次
1.  **Layer 0 (-100)**: 天空盒、雷射背景。
2.  **Layer 1 (-50)**: 中景陰影、背景地標。
3.  **Layer 2 (0)**: 主地形。

## 3. 視覺一致性方針
*   幾何導向：追求數位幾何向量風。
*   霓虹對比：黑色背景與發光線條的冷暖錯覺。
