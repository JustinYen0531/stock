# 視覺背景優化框架 (Visual Background Optimization Framework)

## 1. 核心目標
建立具備深邃立體感的股市數據滑行場景。

## 2. 分層視差標準 (Parallax Standards)
*   **遠景 (Far Vista)**: 捲動速度 = `0.1x`。負責：星辰、極光。
*   **中景 (Mid Range)**: 捲動速度 = `0.35x`。負責：摩天大樓、雷射巨塔。
*   **主地形 (Terrain)**: 捲動速度 = `1.0x`。

## 3. AI 資產規範 (Asset Gen Specs)
*   風格：Digital Vector Style (數位向量風)。
*   細節：低多邊形 (Low-poly) 與霓虹高發光線條。
*   主題：Futuristic Digital Landscape。
