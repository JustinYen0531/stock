# GOOGL source images

Original AI-generated images for the Google / Alphabet high-detail ski theme live here.

The game loads processed assets from `frontend/assets/themes/GOOGL/`:

- `vista.png`: 16:9 parallax background vista.
- `texture.png`: square repeatable mountain terrain material.
- `props/`: transparent PNG element crops for in-terrain and floating props.

Background prompt:

Create a 16:9 high-detail game background asset for a stock-market ski game, Google / Alphabet search and AI platform theme. No skier, no people, no foreground character, no readable text, no UI overlay. The image should be a clean parallax vista layer: distant snowy mountains, soft atmospheric sky, floating cloud islands, search-inspired abstract gateways, colorful data crystals, subtle AI network constellations, frosted glass, polished white ceramic, icy blue light, and restrained red yellow green blue accents. Keep the center and lower-middle area visually open and readable so a separate playable ski slope can be drawn on top by the game engine. Do not place candlestick bars directly on the main path; use them only as tiny distant holographic accents. Side-scrolling friendly composition, wide horizontal layout, moderate detail density, clear depth layers, soft lighting, playful but premium platform-tech mood, not advertising-like, no giant logo, no readable brand marks, no large text.

Terrain prompt:

Create a square high-detail seamless terrain texture for a stock-market ski game, Google / Alphabet search and AI platform theme. Snowy mountain material viewed from above, translucent search-path channels, colorful data crystal inlays, frosted glass, polished white ceramic, icy blue base light, restrained red yellow green blue accents, small network-node glow embedded in the snow. No people, no logos, no readable text, no UI. Must work as a repeatable clipped mountain fill texture, detailed but not noisy, with enough snow and ice readability for a ski slope.

Element prompt:

Create a transparent-background sprite sheet of high-detail Google / Alphabet themed ski-game props. Include colorful data crystals, search-curve track pieces, cloud network orbs, rainbow glass arches, floating island modules, neural globes, data platforms, and small abstract AI nodes. No people, no readable text, no UI, no logos. Each element should be isolated with space between objects, premium glass, polished white ceramic, icy light, soft shadows, and game-ready side-view/isometric readability.
