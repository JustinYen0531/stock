# MSFT source images

Original AI-generated images for the Microsoft high-detail ski theme live here.

The game loads processed assets from `frontend/assets/themes/MSFT/`:

- `vista.png`: 16:9 parallax background vista.
- `texture.png`: square repeatable mountain terrain material.
- `props/`: transparent PNG element crops for in-terrain and floating props.

Background prompt:

Create a 16:9 high-detail game background asset for a stock-market ski game, Microsoft / enterprise cloud software theme. No skier, no people, no foreground character, no readable text, no UI overlay. The image should be a clean parallax vista layer: distant snowy mountains, soft atmospheric sky, subtle futuristic cloud campuses, glass-and-aluminum server architecture, translucent window-like panels, Azure-inspired icy blue cyan light, restrained magenta and green accents, and faint market-data glow in the far background only. Keep the center and lower-middle area visually open and readable so a separate playable ski slope can be drawn on top by the game engine. Do not place candlestick bars directly on the main path; use them only as tiny distant holographic accents. Side-scrolling friendly composition, wide horizontal layout, moderate detail density, clear depth layers, soft lighting, elegant enterprise-tech mood, premium but not advertising-like, no giant software logos, no readable brand marks, no large text.

Terrain prompt:

Create a square high-detail seamless terrain texture for a stock-market ski game, Microsoft / Azure enterprise cloud theme. Snowy mountain material viewed from above, frosted glass pathways, polished aluminum edges, subtle server-grid geometry, translucent window-panel inlays, icy cyan glow, restrained blue-white palette with tiny orange, green, and magenta light accents. No people, no logos, no readable text, no UI. Must work as a repeatable clipped mountain fill texture, detailed but not noisy, with enough snow and ice readability for a ski slope.

Element prompt:

Create a transparent-background sprite sheet of high-detail Microsoft / enterprise cloud themed ski-game props. Include frosted glass walls, cloud data orbs, Copilot-like abstract arches without logos, cyan data elevators, signal towers, small cloud campus structures, Azure-like platforms, server cores, and market chart glass panels. No people, no readable text, no UI, no logos. Each element should be isolated with space between objects, premium glass, polished aluminum, icy cyan lighting, soft shadows, and game-ready side-view/isometric readability.
