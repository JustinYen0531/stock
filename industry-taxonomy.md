# Stock Ski 產業主題分類表

這份檔案是 ticker 對主題映射的細分產業準則來源。

搭配 [four-part-framework.md](four-part-framework.md) 一起使用：

- 這裡負責細分產業、色彩錨點、物理語氣與情緒關鍵字。
- `four-part-framework.md` 負責把它組裝成 `Environment Biome`、`Company Props`、`The Mixer` 與 `Implementation Plan`。
- 裡面的類別與例子是高品質示範，不是封閉清單；若情境需要，可以沿用同樣原則擴寫新的遠景與組件。

## 主題類別

### 1. Digital Aurora / 數位極光 / Tech
- 適用於：軟體、半導體、雲端、AI、網路平台、資安、消費電子。
- 代表色：`#00F2FF`
- 遠景 biome：帶有電路紋理的冰原與極光天幕。
- 中景 props：巨大 GPU 板塊、發光伺服器、光纖樹。
- 操作手感：輕盈、加速度累積快、容錯低。
- 情緒關鍵字：帶電、晶體感、高頻、精密發光。

### 2. Steel Heavyworks / 鋼鐵重工 / Industrial
- 適用於：機械、航太、重工製造、國防、原物料、工業設備。
- 代表色：`#808080`
- 遠景 biome：霧霾工業區、鋼鐵峽谷、灰燼空氣。
- 中景 props：起重機、齒輪、貨櫃山、煙囪。
- 操作手感：慣性重、滾輪轉動需求大、回應偏慢。
- 情緒關鍵字：巨大、粗礪、壓力、機械重量。

### 3. Chicago Vault / 芝加哥金庫 / Finance
- 適用於：銀行、保險、券商、交易所、支付網路、資產管理。
- 代表色：`#FFD700`
- 遠景 biome：摩天樓剪影與古典金融石造建築。
- 中景 props：巨大金幣、石柱、保險庫門、牛頭雕像。
- 操作手感：阻力適中、穩定、重心平衡感強。
- 情緒關鍵字：權威、拋光、穩固、秩序感。

### 4. Neon Media / 霓虹傳媒 / Media
- 適用於：娛樂、串流、廣告、遊戲、出版、創作者平台、社群注意力業務。
- 代表色：`#FF00FF`
- 遠景 biome：賽博街巷、數位噪點、訊號干擾。
- 中景 props：像素塊、對話框、虛擬耳機、故障面板。
- 操作手感：混亂、重力會隨波動改變。
- 情緒關鍵字：躁動、訊號破碎、過度刺激、流行飽和。

### 5. Green Range / 綠能山脈 / Energy
- 適用於：石油、天然氣、再生能源、太陽能、風能、電池儲能、探勘、能源硬體。
- 代表色：`#00FF00`
- 遠景 biome：荒野雪原、風機群、鑽井剪影、外露天候。
- 中景 props：風力發電機、鑽油井、電池槽、巨型太陽能板。
- 操作手感：逆風強、上坡吃力、整體阻力明顯。
- 情緒關鍵字：元素感、暴露感、風切、發電尺度。

### 6. Life Laboratory / 生命實驗室 / Pharma
- 適用於：生技、製藥、診斷、醫療實驗室、藥物研發、健康科學。
- 代表色：`#FFFFFF`
- 遠景 biome：純白實驗室與蜂巢研究艙。
- 中景 props：巨大試管、DNA 雙螺旋、藥錠塔、無菌玻璃。
- 操作手感：精密、hitbox 邊界極嚴。
- 情緒關鍵字：潔淨、臨床、脆弱、嚴謹。

### 7. Grand Luxe / 豪華精品 / Luxury
- 適用於：精品時尚、珠寶、高端美妝、奢華生活方式、頂級配件。
- 代表色：`#D4AF37`
- 遠景 biome：鍍金宮殿、黑色大理石大廳、反射拋光空間。
- 中景 props：鑽石尖晶、絲絨緞帶、香水瓶、金屬飾邊。
- 操作手感：摩擦極低、速度極快、滑行絲滑。
- 情緒關鍵字：奢華、精緻、光澤、排他性。

### 8. Consumer Arcade / 零售超市 / Consumer
- 適用於：超市、電商零售、民生用品、食品零售、折扣連鎖、一般購物。
- 代表色：`#FF9900` (Amazon Orange)
- 遠景 biome：繽紛櫥窗街道與暖色商業照明。
- 中景 props：購物車、包裹、巨大條碼、貨架方塊。
- 操作手感：跳躍節奏強，坡道穿插大量小障礙。
- 情緒關鍵字：忙碌、親切、堆疊、衝動消費。

### 9. Freight Harbor / 物流空港 / Logistics
- 適用於：航運、貨運、快遞、航空、倉儲、港口、配送網路。
- 代表色：`#000080`
- 遠景 biome：海港霧氣、貨櫃碼頭、跑道輪廓。
- 中景 props：集裝箱、飛機翼、巨錨、裝卸起重機。
- 操作手感：轉彎半徑大、慣性強、需要預判。
- 情緒關鍵字：運輸、吞吐量、距離感、重量移動。

### 10. Future Transit / 未來交通 / Auto
- 適用於：車廠、電動車品牌、移動平台、充電網路、自駕、動力系統供應商。
- 代表色：`#A9A9A9`
- 遠景 biome：高速光軌、隧道流光、流線走廊。
- 中景 props：輪胎、充電樁、懸浮引擎、導流鰭片。
- 操作手感：下坡速度加成最強、整體流線感最明顯。
- 情緒關鍵字：俐落、動能、金屬感、速度優先。

### 11. Civic Grid / 基礎建設 / Utility
- 適用於：電網、水務、瓦斯配送、通訊基建、公共事業網路。
- 代表色：`#B8860B`
- 遠景 biome：夕陽荒野、交錯電線杆、高塔、管線、雷達輪廓。
- 中景 props：高壓電塔、天線、水管、網路中繼節點。
- 操作手感：滾輪輸入帶有細微震顫，整體穩定但帶持續抖動。
- 情緒關鍵字：基建感、接地、低鳴、服務關鍵。

### 12. Blueprint Estate / 虛擬地產 / Estate
- 適用於：REITs、房地產開發商、建設導向地產、土地開發、住宅案。
- 代表色：`#F4A460`
- 遠景 biome：施工骨架、藍圖背景、未完工建築群。
- 中景 props：磚塊、捲尺、鷹架梁柱、半完成塔樓。
- 操作手感：負載感逐步增加，落地不均，動能容易被結構邊緣打斷。
- 情緒關鍵字：施工中、模組化、塵土、測繪感。

## 選擇 heuristics

- 即使公司橫跨多個領域，也只選一個主類。
- 如果兩類都說得通，問自己哪個業務最能決定玩家對這檔股票的第一印象。
- 高級車廠優先用 `Auto` 而不是 `Luxury`，因為遊戲幻想核心是速度，不是精品陳列。
- 生技平台優先用 `Pharma` 而不是 `Tech`，因為遊戲幻想核心是精密與邊界判定。
- 運輸營運商優先用 `Logistics` 而不是 `Industrial`，因為遊戲幻想核心是路線與慣性。
- 以持有或開發不動產為主的公司，用 `Estate` 而不是 `Industrial`。

## 主題包欄位

當任務需要結構化輸出時，至少回傳：

- `ticker`
- `company_name`
- `industry_class`
- `industry_label_local`
- `color`
- `biome`
- `props`
- `physics`
- `tone_cues`
- `mapping_reason`

## Physics 詞彙

在寫 config 或設計說明時，優先使用這些遊戲化欄位：

- `acceleration_gain`
- `drag`
- `inertia`
- `friction`
- `gravity_shift`
- `turn_radius`
- `hitbox_strictness`
- `obstacle_density`
- `wind_resistance`
